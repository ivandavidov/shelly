# Шелли Груп АД — Финансов дашборд

Инвеститорски финансов дашборд за **Шелли Груп АД** (BSE: SLYG · XETRA: SLYG).

Покрива 16 тримесечия консолидирани финансови отчети по МСФО: **Q1 2022 → Q4 2025**.

## Бърз старт

```bash
cd docs/
python3 -m http.server 8090
# Отвори http://localhost:8090
```

## Какво включва

- **23 графики** — приходи, маржове, растеж г/г, парични потоци, FCF, баланс, нетен дълг, ликвидност, DuPont ROE, оборотен капитал, дивиденти, EPS, BVPS и др.
- **3 таблици** — Отчет за доходите, Баланс, Парични потоци
- **3 режима** — Тримесечно (16 точки) · Годишно (4 години) · Последни 4 тримесечия (LTM)
- **Акционери и дъщерни дружества** — информационен блок
- **CSV export** на текущия изглед

## Структура

```
docs/
├── index.html            ← HTML на дашборда
├── app.js                ← цялата логика (Chart.js, таблици, изчисления)
├── styles.css            ← стилове
├── generate_quarterly.py ← генерира JSON от ръчно въведени данни
└── data/
    └── shelly_group_YYYY_QN.json   (16 файла, ~100KB всеки)
```
